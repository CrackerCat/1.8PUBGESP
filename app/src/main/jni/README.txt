Build daemon click "BuildLibs.bat" output see in libs/armeabi-v7a folder
Easily moved automatic daemon files to assets click MoveDaemonToAssets.bat

Example build step:
# Without UPX Compress
1. Click BuildLibs.bat
2. Click MoveDaemonToAssets.bat

# Wit UPX Compress
1. Click BuildLibs.bat
2. Click Compress.bat
3. Click MoveDaemonToAssets.bat

## IF ESP NOT WORKED USE UPX COMPRESS, TRY BUILD WITHOUT UPX COMPRESS
